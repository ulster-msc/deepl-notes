Question 9 — Convolution with a 3×3 vertical edge kernel

- Region (rows 1–3, cols 1–3):
  [ [5, 0, 8],
    [3, 1, 6],
    [0, 2, 2] ]

- Kernel (vertical edge detector):
  [ [1, 0, -1],
    [1, 0, -1],
    [1, 0, -1] ]

- Convolution sum (correlation orientation):
  5·1 + 0·0 + 8·(−1) + 3·1 + 1·0 + 6·(−1) + 0·1 + 2·0 + 2·(−1)
  = 5 − 8 + 3 − 6 − 2 = −8

- Absolute value of the result: 8
