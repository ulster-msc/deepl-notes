Question 15 — Compute z and ŷ(z)

Given
- b₀ = 0.5, w₀ = 1 (bias is added as `b₀·w₀`)
- x = [2, 1, 0, 1]ᵀ
- w = [0.2, −0.1, 0.3, 0.4]ᵀ

Compute the pre‑activation z
- Weighted sum with bias: z = b₀·w₀ + x·w
- x·w = 2·0.2 + 1·(−0.1) + 0·0.3 + 1·0.4 = 0.4 − 0.1 + 0 + 0.4 = 0.7
- Add bias: z = 0.5 + 0.7 = 1.2

Activation (sigmoid)
- ŷ = σ(z) = 1/(1 + e^{−z})
- ŷ = σ(1.2) ≈ 0.7685

Final numeric answers
- z = 1.2
- ŷ ≈ 0.7685
