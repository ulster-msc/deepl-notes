Question 13 — Identify layer names

From the code fragment, in order:
1) `Conv2D(32, (3, 3), activation='relu', input_shape=(28, 28, 1))` → Convolutional layer (Conv2D)
2) `MaxPooling2D((2, 2))` → Max pooling layer
3) `Flatten()` → Flatten layer (converts feature maps to a 1‑D vector)
4) `Dense(64, activation='relu')` → Fully connected (Dense) hidden layer
5) `Dense(10, activation='softmax')` → Fully connected (Dense) output layer with softmax
